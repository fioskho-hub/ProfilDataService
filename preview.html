Vue d'ensemble

ProfilDataService est un système complet pour gérer les profils des joueurs sur Roblox.

  • Création automatique des profils à l'arrivée des joueurs.
  • Sauvegarde sécurisée avec retry et backup.
  • Support pour leaderstats, inventaire, PV/Stamina, monnaie, etc.
  • MetaTags et GlobalUpdates pour fonctionnalités avancées.

Structure d’un profil

profile = {
  _Player = Player,
  _Key = "Player_123456",
  Data = {
    Stats = { Level = 1, XP = 0, Cash = 0 },
    Points = { AllPoints = 0, Melee = { Stamina = 50 }, Defense = { Pv = 100 } },
    Inventory = { Items = {} }
  },
  MetaData = { 
    ProfileCreateTime, 
    SessionLoadCount, 
    ActiveSession = {placeId, jobId}, 
    MetaTags, 
    MetaTagsLatest 
  },
  UserIds = [123456],
  GlobalUpdates = {...}
}

Exemples pratiques

1. Leaderstats

local ProfilDataService = require(game.ServerScriptService.ProfilDataService)

game.Players.PlayerAdded:Connect(function(player)
    local profile = ProfilDataService:GetProfile(player)
    profile:Reconcile({
        Stats = { Level=1, XP=0, Cash=0 },
        Points = { AllPoints=0, Melee={Stamina=50}, Defense={Pv=100} }
    })

    local leaderstats = Instance.new("Folder")
    leaderstats.Name = "leaderstats"
    leaderstats.Parent = player

    for statName, value in pairs(profile.Data.Stats) do
        local stat = Instance.new("IntValue")
        stat.Name = statName
        stat.Value = value
        stat.Parent = leaderstats

        stat:GetPropertyChangedSignal("Value"):Connect(function()
            profile.Data.Stats[statName] = stat.Value
            profile:Save()
        end)
    end
end)

2. Inventaire simplifié

local ProfilDataService = require(game.ServerScriptService.ProfilDataService)

game.Players.PlayerAdded:Connect(function(player)
    local profile = ProfilDataService:GetProfile(player)
    profile:Reconcile({ Inventory = { Items = {} } })

    local function AddItem(itemName)
        table.insert(profile.Data.Inventory.Items, itemName)
        profile:Save()
    end

    AddItem("Sword")
    AddItem("Shield")
end)

3. PV/Stamina

profile.Data.Stats.Level = 5
profile:Save()

-- Si vous utilisez StaminaPv
StaminaPv.FillPlayer(player)

Exemples avancés : MetaTags & GlobalUpdates

1. MetaTags

local profile = ProfilDataService:GetProfile(player)

-- Définir un tag pour indiquer un événement
profile:SetMetaTag("LastQuest", "DragonSlayer")
print(profile:GetMetaTag("LastQuest")) -- "DragonSlayer"

-- Le MetaTag est sauvegardé automatiquement
profile:Save()

2. GlobalUpdates

local profile = ProfilDataService:GetProfile(player)

-- Ajouter une update globale (ex: récompense temporaire)
local updateId = profile.GlobalUpdates:AddActiveUpdate({Reward=100, Expire=os.time()+3600})

-- Vérifier toutes les updates actives
for _, upd in pairs(profile.GlobalUpdates:GetActiveUpdates()) do
    print(upd.id, upd.data.Reward)
end

-- Supprimer une update spécifique
profile.GlobalUpdates:ClearActiveUpdate(updateId)

-- Supprimer toutes les updates
profile.GlobalUpdates:ClearAll()

Signaux

  • CorruptionSignal : déclenché si un profil est corrompu
  • CriticalStateSignal : problème critique de sauvegarde
  • MetaTagsUpdated : déclenché quand un MetaTag est mis à jour

Méthodes principales

  • GetProfile(player) : crée ou récupère un profil joueur
  • Release(player) : sauvegarde et libère le profil
  • profile:Reconcile(template) : complète les données manquantes
  • profile:Save() : ajoute le profil à la file d’attente pour sauvegarde
  • profile:OverwriteAsync() : sauvegarde immédiate
  • profile:SetMetaTag(tag,value) : ajoute ou modifie un MetaTag
  • profile:ClearGlobalUpdates() : supprime toutes les GlobalUpdates

Autosave & Backup

Les profils sont sauvegardés automatiquement toutes les 60 secondes et lors du départ du joueur.

Chaque profil a un backup dans BackupStore pour prévenir la perte de données.

local profile = ProfilDataService:GetProfile(player)
profile:Save() -- sauvegarde manuelle
